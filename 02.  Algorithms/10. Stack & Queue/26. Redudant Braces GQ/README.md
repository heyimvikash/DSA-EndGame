# **[Redundant Braces](https://www.interviewbit.com/problems/redundant-braces/)**
## **TC = O(N)**
```java
public class Solution {
    public int braces(String s) {
        Stack<Character> st=new Stack<>();
        for(char c: s.toCharArray()){
            if(c==')'){
                if(st.peek()=='(') return 1;

                char t=st.pop();
                if(st.peek()=='(') return 1;
                st.push(t);
                
                while(!st.isEmpty() && st.peek()!='(') st.pop();
                st.pop();
            }
            else{
                st.push(c);
            }
        }
        return 0;
    }
}
```
